# Definition of Done - not-gta-1 (web app)

## Every change
- [ ] Preflight ran clean at session start (`scripts/preflight.sh`) and repo identity, branch, worktree, dirty state, and shape fingerprint were known.
- [ ] `npm run check` is green.
- [ ] `npm run test` is green; changed behavior has a test that would have caught it.
- [ ] `npm run build` is green.
- [ ] Security, auth, storage, and migration changes name what was verified and how.
- [ ] User-facing changes are visually checked with a screenshot or live browser pass.

## Always
- [ ] No commit, merge, or push without Daniel's word.
- [ ] Evidence for substantive work is captured in `docs/`.
